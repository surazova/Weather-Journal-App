# Project #3: Weather Journal App

## Project Summary

Create a web application that utilizes the OpenWeatherMap API to retrieve weather based on a user's zipcode and dynamically update the User Interface accordingly.

## Languages Used

- HTML
- CSS
- JavaScript

## Node Modules Used

- Express
- BodyParser
- Cors
